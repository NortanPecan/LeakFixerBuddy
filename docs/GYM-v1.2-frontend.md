## F5. Пост-тренировочный диалог

**Статус:** ✅ Done

**Задача:** После сохранения факта — спросить про nextWeight.

**Флоу:**
1. Пользователь нажимает "Завершить тренировку"
2. Показываем диалог с упражнениями и кнопками оценки
3. Три кнопки для каждого упражнения:
   - **😊 Легко** → +2.5 кг (или шаг из шаблона)
   - **😐 Норм** → без изменений
   - **😫 Тяжело** → -2.5 кг (или шаг из шаблона)
4. Превью следующего веса отображается в реальном времени
5. Блок доп. активностей: можно добавить ходьбу, пресс, планку, велосипед
6. При сохранении отправляется POST на `/api/gym/today`

**Где:** После кнопки "Завершить тренировку" в диалоге деталей тренировки

**Реализация:**
- Добавлен диалог `showPostWorkoutDialog`
- Состояние `exerciseRatings` хранит оценку для каждого упражнения
- Состояние `editingActivities` для доп. активностей
- Функция `finalizeWorkout` отправляет данные на API
